# React TypeScript Course Projects

A collection of React and TypeScript projects built throughout my learning journey. This repository contains practical examples, hands-on exercises, and small projects created while studying modern React development with Vite and TypeScript.

## About

The purpose of this repository is to document my progress in React development by building reusable components, learning modern React concepts, and applying TypeScript best practices through real examples.

Topics covered include:

* React Components
* JSX
* Props
* TypeScript Interfaces
* Default Props
* Optional Props
* Conditional Rendering
* Children Props
* State Management
* Event Handling
* Forms
* Hooks
* Routing
* API Integration
* Project-Based Examples

More examples and projects will be added as the course progresses.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/hafizecim/react-typescript-course-projects.git
```

Navigate to the project directory:

```bash
cd react-typescript-course-projects
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
